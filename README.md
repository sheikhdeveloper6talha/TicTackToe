Tag Game Project

An interactive Tag Game built using HTML, CSS, and JavaScript. This project is based on matrix/grid logic where players interact within a structured game layout. The game includes dynamic movement, user interaction, and responsive design to create a fun and engaging experience.

Features
Interactive Gameplay
Matrix/Grid Based Logic
Dynamic Player Movement
Responsive Game Layout
Real-Time User Interaction
Clean UI Design
JavaScript Game Functionality
Technologies Used
HTML5
CSS3
JavaScript
Purpose

The purpose of this project was to improve my JavaScript logic-building skills, understand matrix-based game mechanics, and practice DOM manipulation and interactive frontend development
